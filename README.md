# 🌐 Agrinho 2026

Este projeto é um site simples que apresenta textos explicativos sobre:
- **Agropecuária Sustentável**: como a tecnologia pode ajudar a reduzir desperdícios e aumentar a produtividade de forma responsável.
- **Acessibilidade**: como ferramentas digitais promovem inclusão e autonomia para pessoas com deficiência.

---

## 📂 Estrutura do Projeto

- `index.html` → Página principal do site  
- `style.css` → Estilos visuais (cores, layout, tipografia)  
- `script.js` → Funções JavaScript (alternar entre tema claro e escuro)
